概要

MicroEngine メールフォーム をご利用いただき、誠にありがとうございます。 このソフトウェアは、Webサイトのお問い合わせフォームなどで利用できるメールフォームスクリプトです。

公式サイト：http://microengine.jp/mailform/

使用許諾契約書

本ソフトウェアを使用するには、公式サイト上の使用許諾契約書に御同意頂く必要があります。

インストール方法

オンラインマニュアル（http://microengine.jp/mailform/manual/1.0/）を参照してください。
